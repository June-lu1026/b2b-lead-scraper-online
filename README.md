# 全球骑行配件 B2B 线索采集工具 v6.6 稳定筛选版

## 本版修复

- 不再使用前端点击脚本启动，使用浏览器原生 GET 表单提交，避免按钮无反应。
- 搜索请求和官网抓取均有超时，避免一直转圈。
- 国家/市场、城市/地区真正参与：
  - 搜索词构造
  - 结果排序
  - 最终筛选
- 增加搜索诊断：候选网站、排除后、地区预筛后、已分析、最终结果、实际搜索词。
- 支持“相关优先”和“严格筛选”。
- 支持手动粘贴官网列表，不依赖公开搜索。

## 部署到 Render

Build Command 留空，Start Command 使用：

```bash
npm start
```

## 使用建议

免费版不调用 Google Maps/Places，公开搜索源有时会被服务器限制。若公开搜索结果少，建议：

1. 地区筛选选择“相关优先”。
2. 最低总分、最低匹配度设为 0。
3. 不勾选“只看有邮箱/只看有电话”。
4. 直接粘贴官网列表，一行一个，让工具只做邮箱和电话提取。
