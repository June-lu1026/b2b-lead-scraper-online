# 全球骑行配件 B2B 线索采集工具 v6.5

无需 Google Places API。通过公开搜索源或手动官网列表获取候选网站，并分析公开页面中的邮箱、电话、客户类型与地区匹配度。

## Render 部署

- Build Command: 留空或 `npm install`
- Start Command: `npm start`

## v6.5 更新

- 国家/市场、城市/地区纳入正式筛选条件
- 增加地区筛选模式：优先 / 严格
- 排除关键词/域名改成可编辑，并解释默认排除原因
- 默认排除搜索引擎、社媒、新闻测评、论坛、百科、平台页
- 支持手动官网列表优先分析
- 纯服务端表单提交，按钮不依赖前端 JS
