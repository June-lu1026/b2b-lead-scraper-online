# 全球骑行配件 B2B 线索采集工具 v7.0

本版将国家/城市筛选改为单选下拉框：一次只选择一个国家/市场和一个城市/地区，不再支持多选。

## 特点

- 国家/市场单选
- 城市/地区单选
- 地区筛选模式：不限制 / 相关优先 / 严格匹配
- 支持手动粘贴官网列表
- 支持公开搜索候选网站
- 支持邮箱/电话提取
- 支持 CSV 下载

## 部署

上传并覆盖 GitHub 仓库里的：

- server.js
- package.json
- render.yaml
- README.md

然后 Render 里执行：Manual Deploy → Clear build cache & deploy。
